# Auto-Claim-Kado-IDV

Macro untuk auto claim kado di game Identity V (IDV).

## Fitur
- **Hold E** setiap 3 detik secara otomatis
- Toggle Start/Stop dengan hotkey **F6**
- Program tetap berjalan sampai ditutup manual

## Cara Pakai

### Menggunakan Python
1. Install dependency:
   ```
   pip install keyboard
   ```
2. Jalankan script:
   ```
   python auto_claim_kado.py
   ```

### Menggunakan EXE
1. Download `auto_claim_kado.exe` dari [Releases](https://github.com/UnControl4ble/Auto-Claim-Kado-IDV/releases)
2. Jalankan sebagai **Administrator** (klik kanan â†’ Run as administrator)

## Hotkeys
| Hotkey | Fungsi |
|--------|--------|
| **F6** | Start / Stop macro |
| **X** atau **Ctrl+C** | Tutup program |

## Catatan
- Jalankan sebagai **Administrator** supaya bisa mengirim key press ke game
- Macro akan menekan dan menahan tombol E selama 3 detik, lalu lepas sebentar dan tahan lagi
